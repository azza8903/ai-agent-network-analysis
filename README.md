# AI Agent Network Analysis

This project investigates the network-level behavior of LLM-based AI agents, including traffic patterns, protocol usage, and potential scalability issues.

## Goal

To analyze how agents like Qwen, LLaMA 3, and DeepSeek behave when tasked with real-world actions that require external communication.

## Methodology

- Prompt: *"Get the current weather for London and recommend activities based on the conditions."*
- Tools:
  - `tcpdump` for traffic capture
  - Jupyter Notebook for analysis
- Environment:
  - Mac host connected to a remote server
