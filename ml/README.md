# ML Directory

This directory contains machine learning components and utilities for the AI Agent Network Analysis project.  It adopts code from [Robust Figerprinting](https://github.com/robust-fingerprinting/RF), and puts it in our process.

## Purpose
Houses ML models, training scripts, data processing pipelines, and related utilities for fingerprinting AI agent network behavior.

## Contents
- Model implementation
- Training code
- Test code

## Getting Started
Run `../analysis/pcap2mtam.ipynb` to populate `../datasets`
